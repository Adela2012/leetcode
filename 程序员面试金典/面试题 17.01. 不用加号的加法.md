# 面试题 17.01. 不用加号的加法

设计一个函数把两个数字相加。不得使用 + 或者其他算术运算符。

示例:
```
输入: a = 1, b = 1
输出: 2
```

提示：
```
a, b 均可能是负数或 0
结果不会溢出 32 位整数
```

# 解题
两数相加位运算相当于 `无进位结果a^b` + `进位结果(a&b) << 1`，所以重复不断更新 `无进位结果` + `进位结果`， 最后当 `进位结果` 变为 0 时， 此时 `无进位结果` 就是答案

```ts
function add(a: number, b: number): number {
    while(b) {
        const c = (a & b) << 1
        a ^= b
        b = c
    }
    return a
};
```
